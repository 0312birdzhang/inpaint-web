# [Inpaint-web](https://github.com/lxfater/inpaint-web)

A free and open-source inpainting tool powered by webgpu and wasm

基于Webgpu技术和wasm技术的免费开源inpainting工具

## Demo

https://inpaintweb.lxfater.com/

## Setup

`npm install`

## Development

`npm run dev`
